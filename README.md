## 🧭 Engineering Principles

```
  Own it end-to-end     →  From architecture decision to production alert — full accountability
  Security by default   →  Policy-as-code, least-privilege IAM, shift-left scanning
  Automate the toil     →  If I do it twice, I automate it. If I automate it, I document it.
  Measure what matters  →  MTTR, deployment frequency, change failure rate — SRE without SLOs is just on-call
  Fail safely           →  Canary releases, circuit breakers, auto-rollback — resilience is designed, not hoped for
```
